Overview

Norman’s Action Cycle and the GOMS model are both used in Human–Computer Interaction to analyze user behavior. However, they focus on different levels of interaction and analysis.

Comparison Table

Comparison of Norman’s Action Cycle and GOMS Model
Aspect Norman’s Action Cycle GOMS
Focus User cognition and interaction stages Task execution and user performance
Level of Detail High-level conceptual model More detailed task-oriented model
Main Components Goals, actions, feedback, and evaluation Goals, operators, methods, selection rules
Usage Understanding user-system interaction problems Predicting and analyzing task performance
Planning vs Execution Includes goal formation and evaluation Focuses mainly on execution steps
